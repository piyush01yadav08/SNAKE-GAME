## 👾 About The Game

```bash
> Classic Snake Game reimagined with modern UI and smooth gameplay
> Built using pure JavaScript with animation and dynamic difficulty
```

🎮 This project demonstrates:

* Game loop using `requestAnimationFrame`
* Collision detection logic
* Dynamic speed scaling
* Canvas-based rendering

---

## 🚀 Features

* 🐍 Smooth snake movement
* ⚡ Speed increases over time
* 🌈 Neon glowing snake (random color each game)
* 🟩 Grid-based background
* 🔊 Sound effects (eat + game over)
* ⏸️ Pause / Resume functionality
* 🔄 Restart system
* 📈 Score tracking
